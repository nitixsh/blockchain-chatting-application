# Blockchain Chat Application (ChatBuddy)

A decentralized Web3 Chat Application built using Next.js, Solidity, Hardhat, Ethers.js, and MetaMask.
This DApp enables users to connect their wallet and send secure on-chain messages using the Ethereum blockchain.

---

## 📸 Project Showcase
![Chat App Screenshot](./images/chatapp.png)


## 🌟 Overview
Embark on a journey into the decentralized world of communication with **ChatBuddy**. This platform leverages modern blockchain technologies to ensure user data remains secure, private, and decentralized.

### Key Features:
* **Account Creation**: Users can create profiles on the Ethereum blockchain.
* **Friend System**: Add friends using their wallet addresses.
* **Real-time Chatting**: Secure peer-to-peer messaging.
* **Wallet Integration**: Seamless connection with MetaMask.

---

## 🛠 Tech Stack
* **Frontend**: Next.js & React
* **Smart Contracts**: Solidity ^0.8.24
* **Development Environment**: Hardhat & Ganache
* **Blockchain Interaction**: Ethers.js
* **Wallet**: MetaMask

## 🚀 Getting Started

1️⃣ Clone Repository
git clone https://github.com/nadiasupti/Blockchain-Chat-App.git
cd Blockchain-Chat-App

2️⃣ Install Dependencies
npm install

3️⃣ Start Local Blockchain
If using Ganache CLI:
npx ganache
OR using Hardhat:
npx hardhat node

4️⃣ Deploy Smart Contract
npx hardhat run scripts/deploy.js --network localhost

5️⃣ Run Frontend
npm run dev
